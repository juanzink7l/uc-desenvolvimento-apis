## 🚀 Como usar este repositório


### Faça um fork pelo GitHub:

  1 - Clique no botão "Fork" (canto superior direito da página)

  2 - Escolha sua conta/organização como destino

  3 - Aguarde a cópia ser criada


### Clonar
```
git clone https://github.com/seu-usuario/uc-desenvolvimento-apis.git
```

### Navegar para uma aula específica

``` 
cd uc-desenvolvimento-apis/aula-01-introducao-api
``` 
